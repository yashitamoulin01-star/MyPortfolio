# Yashita Mouli - Portfolio

Personal portfolio for Yashita Mouli, showcasing projects, case studies, and skills across full-stack development and AI engineering.

## Overview
Built to act as a technical engineering hub. It details my work on AI applications, software products, and scalable digital platforms rather than just displaying static screenshots.

## Architecture
This project is built from scratch with zero heavy frameworks to prioritize speed, simplicity, and granular design control:
* **HTML5** for semantic structure
* **Vanilla CSS** with a custom design system utilizing variables and clamp() functions
* **Vanilla JavaScript** for scroll spy and minimal interactions

## Featured Case Studies
* **NeuroSync:** Real-time multimodal communication analysis (PyTorch, FastAPI, Next.js).
* **NewMarket:** B2B wholesale marketplace with complex relational pricing (Next.js, AWS, GraphQL).
* **The Business of Professionals:** Multi-tenant SaaS for independent consultants (Supabase, PostgreSQL, Stripe).
* **FocusSense:** On-device privacy-first focus tracking extension (OpenCV, MediaPipe).
* **Silent Drift:** Distributed real-time whiteboarding engine (WebSockets, Redis).

## Setup
To run this locally, you just need a local server. No build step is required.
1. Clone the repository
2. Run a live server (e.g. VS Code Live Server extension or Python's `python -m http.server`)
3. Open `index.html`
